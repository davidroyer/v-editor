# v-editor

My polished Vue component.

## Install

```bash
yarn add v-editor
```

CDN: [UNPKG](https://unpkg.com/v-editor/) | [jsDelivr](https://cdn.jsdelivr.net/npm/v-editor/) (available as `window.VEditor`)

## Usage

```vue
<template>
  <v-editor></v-editor>
</template>

<script>
import VEditor from 'v-editor'

export default {
  components: {
    VEditor
  }
}
</script>
```
